About TechTrend Store

Welcome to TechTrend Store, a dummy e-commerce project designed for educational and demonstration purposes. This Capstone Project serves as an environment for learning to implement frontend design, user experience optimization, and digital analytics tracking tools like Google Analytics (GA4) and Google Tag Manager (GTM).

Our Mission

Our goal is to simulate a fully functional e-commerce storefront to practice tracking user actions, analyzing traffic patterns, and optimizing the conversion funnel.

Capstone Objectives

  • Implement structural HTML and modern CSS.
  • Enable basic DOM interactions with JavaScript.
  • Provide actionable IDs and classes for easy GTM tracking.
  • Analyze dummy traffic and event triggers via Google Analytics.

Note: None of the products listed on this site are real, and no actual purchases can be made.
